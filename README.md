# Club Fitness Gym Management Database

## 📌 Overview
A relational database system designed for managing gym operations including members, trainers, payments, and workout plans.

## 🧠 Business Problem
Manual systems caused billing errors, duplicate records, and scheduling conflicts.

## 🛠️ Tech Stack
- PostgreSQL
- Python
- Faker (data generation)
- SQL
- VS Code

## 🗄️ Database Features
- Fully normalized schema (3NF)
- Referential integrity constraints
- Analytical queries
- Star schema for BI

## 📊 Sample Analytics
- Revenue per member
- Payment trends
- Membership profitability

## 🚀 How to Run
1. Create PostgreSQL database
2. Run SQL in `/sql/ddl`
3. Seed data using Python scripts
